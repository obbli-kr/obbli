## Obbli Project에 오신 것을 환영합니다.

### Obbli란?

오브리(오블리가토, obbligato)는 음악 용어로, 피아노나 관현악 등의 반주가 있는 독창곡에 독주적 성격을 가진 다른 악기를 더하는 것을 의미합니다. 이는 생략할 수 없는 필수적인 부분을 뜻하며, 음악에서 중요한 지시어로 사용됩니다.

### 프로젝트 소개

Obbli 프로젝트는 한국에서 클래식 연주자들을 통칭하여 부르는 ‘오브리’라는 용어에서 영감을 받았습니다. 한국의 클래식 연주자들 중 대부분은 한 번쯤 ‘오블리’라고 불린 경험이 있을 것입니다. 이 프로젝트의 목표는 이러한 연주자들의 여정을 함께하는 국내 최대의 클래식 연주자 커뮤니티를 구축하는 것입니다.

Obbli는 단순히 연주자들을 위한 구인구직 플랫폼을 넘어, 레슨, 채용공고, 연주회 및 공연 정보, 게시판, 악기 직거래 등 다양한 기능을 제공하여 클래식 음악인들이 소통하고 성장할 수 있는 공간을 마련합니다.

이곳에서 클래식 음악에 대한 열정을 공유하고, 새로운 기회를 발견하며, 함께 음악을 만들어가는 여정을 시작해보세요.

### run the development server

root 폴더에서 다음을 실행

```
// 처음 한 번만 실행
npm i
```

```
npm run dev
```

[http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### lint setting

vscode 내 settings.json에 아래 코드 추가

```
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  }
}
```
